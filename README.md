## 平台说明

### 微信公众号
- **内容类型**: 深度价值文章（1000-3000字）
- **风格**: 专业、权威、教育性
- **目标**: 建立品牌权威，促进订阅和分享

### 小红书
- **内容类型**: 真实种草笔记（200-800字）
- **风格**: 生活化、真实、情感连接
- **目标**: 品牌曝光，用户互动

### 抖音
- **内容类型**: 短视频脚本（50-200字）
- **风格**: 快节奏、娱乐化
- **目标**: 病毒传播，快速触达

### 百家号
- **内容类型**: SEO优化文章
- **风格**: 信息丰富、结构化
- **目标**: 搜索流量，品牌可发现性

### CSDN
- **内容类型**: 技术教程和问题解决
- **风格**: 精准、代码导向
- **目标**: 技术社区影响力

### B站
- **内容类型**: 长视频脚本
- **风格**: 知识性、趣味性
- **目标**: 深度内容，粉丝积累

## 使用规范

### 内容状态管理
- `published/` - 已发布的内容
- `draft/` - 正在创作中的草稿
- `archive/` - 归档的历史内容

### 命名规范
- 格式: `YYYY-MM-DD_标题.扩展名`
- 示例: `2026-04-25_A公司产品发布.md`
